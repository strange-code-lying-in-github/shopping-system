\doxysubsection{Detailed Description}
This class represents a Data Transfer Object (DTO) for the Audit Trail entity. 

It extends the \doxylink{class_base_d_t_o}{Base\+DTO} class and provides a mapping of fields and their types. The \doxylink{class_audit_trail_d_t_o}{Audit\+Trail\+DTO} class also provides a method to retrieve the DTO home instance.
