\doxysubsection{Detailed Description}
The \doxylink{class_app_logging_stream}{App\+Logging\+Stream} class implements the \doxylink{interface_i_app_logging_stream}{IApp\+Logging\+Stream} interface and provides methods for logging messages to an output stream. 

It includes methods for logging strings, objects, and throwables, as well as a method for flushing the output stream.
